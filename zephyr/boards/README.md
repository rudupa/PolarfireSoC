# Zephyr Board Customizations

Document hart-specific board definitions here:

- `mpfs_discovery_u54_1` – Linux-friendly interrupts disabled, OpenAMP shared memory enabled.
- `mpfs_discovery_u54_2` – Radar DMA control tasks.
- `mpfs_discovery_u54_3` – Spare core for experimental algorithms.

Record clock, pinmux, and mailbox settings per hart.
